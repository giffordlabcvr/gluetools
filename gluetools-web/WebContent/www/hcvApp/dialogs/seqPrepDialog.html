Preparing your sequences for analysis

Sequences must be in FASTA format. For each sequence, the FASTA ID must specify the closest related reference sequence in the GLUE project.

One option is to specify the reference name somewhere in the sequence, with dots on either side: 

>Seq912381.REF_3a_X76918.2923.2341

Another option is to use the CVR's FASTA format: 

>13S12_S13_L001_R1_001-Ref.3a.DE.x.HCVCENS1.X76918.fa.sam 1378281

Both the above FASTA IDs specify that the closest reference is REF_3a_X76918.
