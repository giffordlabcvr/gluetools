HCV-GLUE

Overview

The emergence of clinically-relevant hepatitis C virus (HCV) resistance to direct-acting antiviral drugs in parallel with advances in next generation sequencing technology means there is a need for computational resources that allow clinicians to make optimal treatment decisions based on analysis of virus sequence data. These resources are also an important component of efforts to study and monitor HCV genetic variation, and to evaluate its impact on disease course and treatment outcome. HCV-GLUE is interactive online framework for rapid, standardised analysis of HCV sequence data.

Data sources

Data relating to structural, evolutionary and phenotypic aspects of HCV were obtained from various sources. These are integrated together in GLUE, a data-centric software framework for capturing, developing and utilising nucleotide-level virus evolutionary hypotheses. The result is a GLUE project specific to HCV.

Web analysis tool

The HCV-GLUE project is embedded into an interactive sequence analysis tool which operates over the world wide web.
