GLUE

Java implementation of GLUE virus-sequence-management toolkit

View on GitHub Download .zip Download .tar.gz

GLUE is a software package for developing nucleotide-level evolutionary hypotheses for virus and endogenous viral element (EVE) genomes. It enables the dissemination of these hypotheses on the web and the utilisation of the hypotheses in wider software systems.

Design Overview

At the heart of GLUEtools is an entity-relationship model which has been designed for representing:

  • Virus and EVE genome data
  • Phenotypic and other data associated with specific sequences
  • Genomic feature annotations
  • The nucleotide-level proposed evolutionary relationships between genomes
  • Configurable modules for manipulation and analysis of the data

Data hosted inside a GLUEtools instance is organised into independent projects. Typical activities associated with developing a GLUEtools project:

  • Import sequence data into GLUEtools from a public database or elsewhere
  • Define relevant additional sequence data fields and obtaining corrected / normalised values for these fields
  • Filter out certain sequences from the project
  • Select reference sequences and define genomic features on these references
  • Apply sequence alignment techniques to capture homologies between genome segments
  • Arrange data into evolutionary clades and define relationships between these clades
  • Define project-specific analysis modules

Once a project has been developed, it can be published as a web resource, which allows:

  • Dissemination of the project data, for example in conjunction with an academic publication
  • Utilisation of the project for automated sequence analysis

Current GLUE projects

  • HCV-GLUE - a GLUE project for Hepatitis C virus (HCV)
  • [SRLV-GLUE] - a GLUE project for small ruminant Lentivirus (SRLV)
  • [BTV-GLUE] - a GLUE project for bluetongue virus (BTV)
  • [HEV-GLUE] - a GLUE project for hepatitis E virus (HEV)
