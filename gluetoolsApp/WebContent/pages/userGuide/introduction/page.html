This user guide is for bioinformaticians who would like to:
  • Create a new GLUE project for a specific virus or set of related viruses.
  • Modify or extend an existing GLUE project.
  • Extract data from an existing GLUE project.
  • Run some analysis using a GLUE project.

Getting started

Follow the installation guide for step-by-step instructions to get GLUE running on your computer.

Once you have GLUE running, download and build the example GLUE project.

Next, learn how to use the command line interpreter to perform basic GLUE project queries.

After this, try building your own minimal GLUE project.

Further reading

There are various other aspects of GLUE to learn, the importance of each depends on your particular objectives.
