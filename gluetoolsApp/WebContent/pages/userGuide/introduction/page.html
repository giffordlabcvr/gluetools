GLUE provides a powerful command line interface. This user guide is for bioinformaticians who would like to use the GLUE command line interface to:
  • Create a new GLUE project for a specific virus or set of related viruses or EVEs.
  • Modify or extend an existing GLUE project.
  • Extract data from an existing GLUE project.
  • Run some new analysis on an existing GLUE project.

Getting started

Follow the installation guide for step-by-step instructions to get GLUE running on your computer.

Once you have GLUE running, learn how to use the interactive command line to build, manipulate and query the example GLUE project.

When you are ready to create your own GLUE project, familiarise yourself with the core GLUE schema and read about GLUE project structure, and how to manage a GLUE project.

More advanced topics

Commands often use Cayenne queries to select sets of objects within the project.

As well as basic commands, you will need to use various modules to perform more complex operations.

Some modules use Freemarker templates to transform objects into text.
