JPA 2.1 Readme

JSR 338 (distributed under the license on that page) is packaged in an OSGi format for distribution with the EclipseLink Project which is the reference implimentation of the JPA 2.1 specification.

The source code is available within the project's javax.persistence Git repository. The binaries are distributed in OSGi enabled javax.persistence_2.1.*.jar series bundles.
